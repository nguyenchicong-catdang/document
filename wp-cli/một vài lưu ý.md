## lưu ý

> wp option update siteurl "http://localhost:8000/wp-learn"

>  wp option update home "http://localhost:8000/wp-learn"

> wp option add upload_url_path "/wp-learn/wp-content/uploads"

> wp option update upload_url_path "/wp-learn/wp-content/uploads"

```md
Tên Option	Ý nghĩa	Ví dụ giá trị
upload_path	Đường dẫn vật lý (ổ cứng)	/var/www/public/wp-learn/wp-content/uploads
upload_url_path	Đường dẫn URL (trình duyệt)	http://localhost:8000/wp-learn/wp-content/uploads

```
