# Dockerfile "Full Combo" (SQLite + Code + Nginx + PHP)

FROM php:8.2-fpm-alpine

# 1. Cài đặt đủ bộ
RUN apk add --no-cache nginx supervisor sqlite

# 2. Setup thư mục
WORKDIR /var/www/html
COPY . .

# 3. Quan trọng: Tạo file SQLite và cấp quyền cho user web (www-data)
# Giả sử Laravel dùng database/database.sqlite
RUN touch database/database.sqlite && \
    chown -R www-data:www-data /var/www/html/storage /var/www/html/database

# 4. Copy config Nginx/Supervisor (như đã bàn ở trên)
COPY ./docker/nginx.conf /etc/nginx/http.d/default.conf
COPY ./docker/supervisord.conf /etc/supervisord.conf

CMD ["/usr/bin/supervisord", "-c", "/etc/supervisord.conf"]

Để an toàn, khi chạy lệnh Docker cho khách, bạn nên mount nhẹ cái file SQLite đó ra ngoài:

docker run -v ./data.sqlite:/var/www/html/database/database.sqlite ...

## docker/supervisord.conf

```conf
[supervisord]
nodaemon=true
user=root
logfile=/dev/null
logfile_maxbytes=0

[program:php-fpm]
command=php-fpm -F
stdout_logfile=/dev/stdout
stdout_logfile_maxbytes=0
stderr_logfile=/dev/stderr
stderr_logfile_maxbytes=0

[program:nginx]
command=nginx -g "daemon off;"
stdout_logfile=/dev/stdout
stdout_logfile_maxbytes=0
stderr_logfile=/dev/stderr
stderr_logfile_maxbytes=0
```

RUN docker-php-ext-install pdo pdo_sqlite

```Nginx
server {
    listen 80;
    server_name localhost;
    root /var/www/html/public; # Trỏ thẳng vào public như bạn muốn

    index index.php index.html;

    location / {
        try_files $uri $uri/ /index.php?$query_string;
    }

    location ~ \.php$ {
        fastcgi_pass 127.0.0.1:9000;
        fastcgi_index index.php;
        include fastcgi_params;
        fastcgi_param SCRIPT_FILENAME $document_root$fastcgi_script_name;
    }
}
```
